东莞城市学院 2025 招生简章 H5 发布说明

这个文件夹就是可上传发布包。

必须一起上传的文件：
- index.html
- styles.css
- app.js
- 所有“官宣！东莞城市学院2025年招生简章...”图片文件

上传后，请打开平台给你的 https 链接。
把这个链接发到微信聊天里，微信通常会显示标题、简介和封面图链接卡片。

几个常见发布方式的区别：

1. Vercel
- 优点：适合 H5 静态网页，更新方便，速度快，能免费用。
- 注意：通常需要登录账号，国内访问速度有时不稳定。
- 适合：想最快拿到一个可访问链接。

2. GitHub Pages
- 优点：免费、稳定，适合长期放静态网页。
- 注意：需要 GitHub 账号；如果不会用 GitHub，第一次会稍微麻烦。
- 适合：长期保存，不频繁改。

3. 腾讯云/阿里云静态网站
- 优点：国内访问更稳，更适合微信用户打开。
- 注意：可能需要实名认证、购买对象存储或服务器、配置域名和 HTTPS。
- 适合：正式招生宣传、访问量较多、希望国内速度稳定。

建议：
- 先用 Vercel 或 GitHub Pages 快速测试。
- 如果要正式大范围发微信，优先考虑腾讯云或阿里云，并绑定正式域名。

微信卡片提醒：
- 普通网页链接一般显示为“标题 + 简介 + 小封面图”的链接卡片。
- 如果想显示更大的微信小程序卡片，需要另外开发微信小程序。
- 如果想显示公众号文章样式，需要通过公众号发布文章。
